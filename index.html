<del>

Voltar

Definição e uso

A tag <p> define um parágrafo. A <del> tag define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído. Dica: Observe também a tag <ins> para marcar o texto inserido.

<p>

Voltar

Definição e uso

A tag <p> define um parágrafo. Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada <p> elemento. Dica: Use CSS para estilizar parágrafos.

Voltar
